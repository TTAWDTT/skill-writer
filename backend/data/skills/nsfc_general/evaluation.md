# 国家自然科学基金评审标准

## 评审维度与权重

### 1. 科学价值（40%）

**评审要点：**
- 研究问题是否具有重要科学意义
- 是否处于学科前沿
- 是否有望取得突破性进展

**评分标准：**
- 优秀：问题前沿、意义重大、可能产生重要影响
- 良好：问题有价值、有一定前沿性
- 一般：问题较常规、创新性不足
- 较差：问题陈旧、缺乏科学价值

### 2. 创新性（25%）

**评审要点：**
- 研究思路是否新颖
- 研究方法是否有创新
- 是否能推动学科发展

**评分标准：**
- 优秀：思路新颖、方法创新、有望开辟新方向
- 良好：有一定创新、能推进现有研究
- 一般：创新性有限、属于常规研究
- 较差：无明显创新

### 3. 可行性（20%）

**评审要点：**
- 研究方案是否合理
- 技术路线是否可行
- 研究计划是否切实

**评分标准：**
- 优秀：方案严谨、技术成熟、计划合理
- 良好：方案基本合理、技术可行
- 一般：方案有待完善、存在一定风险
- 较差：方案不合理、难以实现

### 4. 研究基础（15%）

**评审要点：**
- 申请人及团队的研究积累
- 前期工作是否支撑本项目
- 研究条件是否具备

**评分标准：**
- 优秀：积累深厚、前期充分、条件完备
- 良好：有一定积累、条件基本满足
- 一般：积累不足、条件有限
- 较差：缺乏相关基础

## 评审人关注点

1. **前期研究基础**
   - 是否有相关领域的研究成果
   - 发表论文的质量和数量
   - 承担过的相关项目

2. **科学问题凝练**
   - 问题是否明确、具体
   - 是否抓住了关键科学问题
   - 问题的创新性和重要性

3. **技术路线设计**
   - 是否清晰、完整
   - 各部分是否有机联系
   - 是否具有可操作性

4. **创新点表述**
   - 是否真正创新
   - 是否有理论或实验支撑
   - 是否能够实现

5. **预期成果合理性**
   - 是否与研究内容匹配
   - 是否可实现
   - 是否有价值

## 常见拒绝理由

- 创新性不足，属于常规研究
- 研究基础薄弱，可行性存疑
- 技术路线不清晰，可操作性差
- 研究内容过于宽泛，聚焦不够
- 文献综述不全面，对领域把握不准
- 预期目标过高，难以实现

## 提升建议

1. **突出创新** - 明确说明与现有工作的区别
2. **强化基础** - 展示前期工作的积累
3. **完善方案** - 技术路线要详细可行
4. **精炼问题** - 科学问题要明确聚焦
5. **合理预期** - 目标要切实可行
