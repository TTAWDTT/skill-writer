# 国家自然科学基金申报书写作规范

## 整体要求

1. **语言规范** - 专业术语使用准确，避免口语化表达
2. **逻辑清晰** - 层次分明，论述有条理
3. **数据翔实** - 用数据和事实支撑观点
4. **创新突出** - 创新点明确，但不过度夸大
5. **格式规范** - 图表清晰，格式统一

## 语言风格

- 使用第三人称或无主语句式（"本项目拟..."而非"我们将..."）
- 避免口语化表达和网络用语
- 减少修饰性词语，多用精确描述
- 使用学术规范的表达方式

## 各部分写作要点

### 摘要
- 控制在400字左右
- 包含：背景、目标、方法、意义四要素
- 语言高度凝练
- 突出创新点

### 立项依据
- 文献综述要全面、深入
- 准确指出现有研究的不足
- 清晰阐述本项目的必要性
- 引用文献要新（近5年为主）

### 研究内容
- 目标要具体、可衡量
- 内容要系统、完整
- 技术路线要清晰、可行
- 各研究内容之间要有逻辑联系

### 可行性分析
- 要有充分的前期基础支撑
- 方法和技术的可行性论证
- 团队能力匹配
- 研究条件保障

## 常见问题

1. **文献综述蜻蜓点水** - 要深入分析，不是简单罗列
2. **创新点表述空泛** - 要有具体支撑和论证
3. **技术路线过于笼统** - 要详细、可操作
4. **研究基础与项目关联弱** - 要明确关联性

## 禁忌

- ❌ 不要照搬他人成果
- ❌ 不要过度承诺无法实现的目标
- ❌ 不要忽视可行性论证
- ❌ 不要使用过多修饰性语言
- ❌ 不要遗漏关键参考文献

## 字数参考

| 部分 | 建议字数 |
|------|---------|
| 摘要 | 300-500 |
| 立项依据 | 2000-4000 |
| 研究内容 | 2500-4000 |
| 可行性分析 | 500-1000 |
| 研究计划 | 300-600 |
| 预期成果 | 200-500 |
| 研究基础 | 1000-2500 |
